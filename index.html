Restaurant Manager

Summary

An interactive website with social signin (Google and Facebook) that helps restaurant staff and owners manage restaurants and their menus. Users can view, create, edit, and delete cuisines, cuisine "base menu items", restaurants, and restaurant menu items, as well as view other users' profiles.

Technical Details

Written in Python (using Flask) on the backend and HTML/CSS/JavaScript on the frontend, with a SQLite database. Performs server-side and both real-time and on-submit client-side form validation, uses Flask blueprints to improve modularization, and provides a Python data access object to separate database implementation from server programming.

Stack Exchange API and Analysis

Summary

Stack Echange API and Analysis is a Java library that provides classes and methods to manipulate, analyze, and visualize data from Stack Exchange, an online question and answer network with a variety of distinct communities dedicated to topics ranging from math to music to the workplace to religion. It also explores, via multiple regression, what makes a Stack Exchange question "useful" (i.e., receive upvotes).

Technical Details

Written in Java. You can download the latest Stack Exchange "data dump" at https://archive.org/details/stackexchange

Port Scheduler

Summary

At seaports, bulk mineral cargo reclaimers stockpile delivered goods and then reclaim the goods onto arriving ships. Port Scheduler is a program that optimizes reclaimer activity subject to nearly real-world constraints (for example, varying schedules of arriving ships with various cargo requirements).

Technical Details

Written in a constraint/logic programming language called MiniZinc. To run this software, you need MiniZinc 2.0.x and the GECODE 4.4.x solver. Both of these are included in the bundled version of the MiniZinc IDE 0.9.9, which you can find at the MiniZinc website.

Swiss Tournament Manager

Summary

Swiss Tournament Manager is a backend system for managing swiss-style tournaments. The database can store standings and calculate next-round pairings for multiple swiss-style tournaments with even or odd numbers of players, matches that end in ties, and multiple tie-breakers.

Technical Details

Written in Python, with a PostgreSQL database. Currently, no frontend.

Route Planner

Summary

Route Planner is a re-creation of Google Maps' path finding capabilities. It can 1) calculate the shortest path in terms of trip duration (or nearly shortest path, if the path is a "Traveling Salesperson Problem") between up to eight locations, and 2) demonstrate the differences between various graph search algorithms.

Technical Details

Written mostly in Java (including the graph data structures and path finding algorithms). The GUI was built mainly in Java using JavaFX and GMapsFX, with a little HTML, JavaScript, and CSS.
