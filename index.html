Restaurant Manager

Summary

An interactive website with social signin (Google and Facebook) that helps restaurant staff and owners manage restaurants and their menus. Users can view, create, edit, and delete cuisines, cuisine "base menu items", restaurants, and restaurant menu items, as well as view other users' profiles.

Technical Details

Written in Python (using Flask) on the backend and HTML/CSS/JavaScript on the frontend, with a SQLite database. Performs server-side and both real-time and on-submit client-side form validation, uses Flask blueprints to improve modularization, and provides a wholly-Python data access object to separate database implementation from server programming.
