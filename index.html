Restaurant Manager

Summary

Restaurant Manager is an interactive website with social signin (Google and Facebook) that helps restaurant staff and owners manage restaurants and their menus. Users can view, create, edit, and delete cuisines, cuisine "base menu items", restaurants, and restaurant menu items, as well as view other users' profiles.

Technical Details

Written in Python (using Flask) on the backend and HTML/CSS/JavaScript on the frontend, with a SQLite database. Performs server-side and both real-time and on-submit client-side form validation, uses Flask blueprints to improve modularization, and provides a Python data access object to separate database implementation from server programming.

Stack Exchange API and Analysis

Summary

Stack Echange API provides classes and methods to manipulate, analyze, and visualize data from Stack Exchange, the popular expert question-and-answer network. The project also explores, via multiple regression, what makes a Stack Exchange question "useful" (i.e., receive upvotes) in different communities.

Technical Details

Written in Java. To get data to use with the application, download a Stack Exchange "data dump" at https://archive.org/details/stackexchange

Port Scheduler

Summary

At seaports, bulk mineral reclaimers stockpile delivered cargo and then reclaim the cargo onto arriving ships. Port Scheduler is a program that optimizes reclaimer activity subject to nearly real-world constraints (for example, varying schedules of arriving ships with various cargo requirements).

Technical Details

Written in a constraint/logic programming language called MiniZinc. To run this software, you need MiniZinc 2.0.x and the GECODE 4.4.x solver. Both of these are included in the bundled version of the MiniZinc IDE 0.9.9, which you can find at the MiniZinc website.

Swiss Tournament Manager

Summary

Swiss Tournament Manager is a backend system for managing swiss-style tournaments. The application calculates standings (with tie-breakers), stats, and next-round pairings for any number of distinct tournaments with the same or different players.

Technical Details

Written in Python, with a PostgreSQL database. Currently, has no frontend.

Route Planner

Summary

Route Planner is a re-creation of Google Maps' path finding capabilities. It can 1) calculate the shortest path in terms of trip duration (or nearly shortest path, if the route is a "Traveling Salesperson Problem") between up to eight locations, and 2) demonstrate the differences between various graph search algorithms.

Technical Details

Written mostly in Java (including the graph data structures and path finding algorithms). The GUI was built mainly in Java using JavaFX and GMapsFX, with a little HTML, JavaScript, and CSS.
